@plato/aws-meta

Introspects AWS EC2 and ECS metadata from running containers.

Usage

 
import { getEC2LocalIPv4 } from "@plato/aws-meta";

try {
    const ip = await getEC2LocalIPv4();
    // Do something with "ip"
} catch (e) {
    // Failed to get local IP
}
